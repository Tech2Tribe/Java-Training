# Java Training 🚀

A collection of Java programs written while self-studying *Introduction to Java Programming* by Y. Daniel Liang (10th Edition).

---

## 📁 Project Structure

### Chapter 1 - Introduction to Java
Basic Java programs covering fundamental concepts such as:
- Displaying messages
- Computing expressions
- Area and perimeter calculations
- Population projection
- Summation of a series
- Cramer's Rule

### Chapter 2 - Elementary Programming
Programs covering variables, data types, and arithmetic operations including:
- Unit conversions (feet to meters, pounds to kilograms, Celsius to Fahrenheit)
- Financial calculations (interest, compound value, tips, investment)
- Physics problems (acceleration, energy, wind chill)
- BMI calculator
- Cost of driving

### Chapter 3 - Selection
Programs covering if/else statements and switch expressions including:
- Lottery and quiz programs
- BMI revision
- Tax computation
- Rock Paper Scissors
- Days in a month
- Triangle and circle geometry

### Chapter 4 - Mathematical Functions, Characters and Strings
Programs covering Math methods, char operations, and String manipulation including:
- Vehicle plate number generator
- Monetary units
- Great circle distance
- Hex and binary conversion
- Vowel/consonant checker
- Birthday guessing game
